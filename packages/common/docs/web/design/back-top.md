### 何时使用

需要一键返回顶部的时候；

一般“返回按钮”都是延迟出现，即当用户进行一定程度的下拉滚动操作时，再让按钮出现。

### 与页面布局相关

当页面（容器）滚动到一定高度的时候，在右下角会出现返回顶部的按钮。

### 建议/慎用示例

一个页面只能有一个“返回顶部”的按钮，不可以在不同节点并存多个。
