### 何时使用

新手引导：针对首次使用产品的用户，介绍产品核心功能、操作流程与界面布局，帮助用户快速上手；

功能引导：在新功能上线或功能更新时，引导用户了解并使用新功能，降低用户认知成本；

任务引导：在用户执行特定任务（如注册、下单、设置等）时，提供分步指导，确保用户顺利完成任务。

### 建议/慎用示例

引导的内容要简洁，突出关键信息，不要在一个引导步骤中塞入过多的文字或功能介绍，以免让用户感到困惑和压力过大。
