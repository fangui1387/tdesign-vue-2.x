### 何时使用

常用于展示一组平级的信息，比如产品 Banner、公告模块等;

当空间不足时，可以用这种模式收纳内容，进行轮播展示;

可以用户主动触发，也可以系统自动轮播。

### 与页面布局相关

一般常见页面顶部（导航之下），通过这种方式可以更好的吸引用户注意力。
